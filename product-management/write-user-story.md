# /write-user-story -- Backlog Item Generator

Break a feature into well-structured user story.

## Invocation
...
/write-user-story Add a dark mode toggle to the settings screen
/write-user-story [paste your notes or bullet points]
/write-user-story [attach a design screen or mockup]
/write-user-story [attach a screen + paste notes]
/write-user-story                                    # asks for feature details interactively
...

## Formats

### User Story
**Format**: "As a [user type], I want [capability], so that [benefit]"
- Follow the 3 C's: Card (brief), Conversation (context), Confirmation (acceptance criteria)
- Validate against INVEST: Independent, Negotiable, Valuable, Estimable, Small, Testable
